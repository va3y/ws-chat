## Start Dev Server

```bash
npm run dev
```

This will start a dev server on port `3000` and will watch for changes.

## Testing Production Build

### Build

```bash
npm run build
```

### Start

```bash
npm run start
```

This will start a production server on port `3000`.

## Deploying To Vercel

### Building

```bash
  vercel build --prod
```
  
### Deploying 

```bash
vercel deploy --prod --prebuilt
```
